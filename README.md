# Console-X-Stone (CXS)

A Gorilla Tag mod that combines console functionality with an admin system - now running entirely on GitHub!

## Configuration

Your mod is configured to use:
- **Admin List**: `Cha554/mist.online` → serverdata.json
- **Asset Bundles**: `Cha554/Console-X-Stone` → ServerData/

## Features

- Admin & Super-Admin system with in-game indicators
- Asset bundle loading from GitHub
- No external server needed
- Easy admin management via JSON file
- Telemetry disabled (privacy-focused)
